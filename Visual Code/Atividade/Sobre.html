HTML

HRML (hyperText Markup Language) é a linguagem de marcação utuilizada para estruturar páginas web.

CSS

CSS (Cascading Style Sheets) é usado para estilizar elementos HTML e definir o layout das páginas.

JavaScript

JavaScript é uma linguagem de programação que permite adicionar interatividade às páginas web.
